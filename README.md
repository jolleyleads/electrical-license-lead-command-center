# Electrical License Lead Command Center

## Overview
Lead intelligence platform for identifying, scoring, and managing high-intent contractor leads seeking licensed electrical contractor coverage.

## What It Does
- Saves contractor leads
- Scores buyer intent
- Ranks leads as Low, Medium, or High priority
- Tracks company, contact, city, state, source, and project need
- Provides recommended next actions
- Exposes Make.com-ready API endpoints
- Supports outreach automation and CRM-style workflow tracking

## Compliance Note
This should be positioned as licensed electrical contractor partnership, qualifying master electrician services, or compliant project coverage. Do not market it as simply renting a license.

## Tech Stack
- Python
- Flask
- SQLite
- Flask SQLAlchemy
- HTML/CSS/JavaScript
- Render
- Make.com-ready APIs

## API Endpoints
GET /api/leads  
POST /api/add-lead  
POST /api/score  
POST /api/outreach-draft  
POST /api/update-lead  
GET /api/health
